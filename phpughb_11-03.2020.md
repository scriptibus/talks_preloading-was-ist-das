# Lexing oder Tokenizing
## Verwandelt einen Code-String zu eine Token-Stream


[.code-highlight: none]
[.code-highlight: 1-3]
[.code-highlight: 5]
[.code-highlight: 7-14]

```php
<?php

$code = '<?php $a = 1 ?>';

$tokens = token_get_all($code);

foreach ($tokens as $token) {
    if (is_array($token)) {
        echo "Line {$token[2]}: ", token_name($token[0]), " ('{$token[1]}')", PHP_EOL;
    } else {
        var_dump($token);
    }
}
```
